# spamham_classifier
Proyek ini bermanfaat untuk meningkatkan efisiensi pengelolaan email dengan menyaring pesan spam secara otomatis, sehingga pengguna dapat fokus pada pesan penting. Sistem ini membantu mengurangi risiko keamanan dari ancaman seperti phishing dan malware, serta meningkatkan produktivitas dengan meminimalkan gangguan. Selain itu, model berbasis Multinomial Naive Bayes ini ringan, cepat, dan dapat diintegrasikan ke dalam layanan email berskala besar. Proyek ini juga menjadi sarana pembelajaran penerapan machine learning untuk kasus nyata dan membuka peluang pengembangan teknologi deteksi teks lebih lanjut.
